# New Zealand Border Movements Analysis

## 1. Business Context
New Zealand border movements play a critical role in tourism recovery, population change, and labour supply.
Following the COVID-19 pandemic, border activity experienced significant disruptions, making it important to understand how travel patterns have evolved in the post-pandemic period.

This project analyses official daily border movement data to identify key trends, seasonal patterns, and net population movements, providing insights that may be relevant for policymakers, businesses, and planners in New Zealand.

---

## 2. Project Objective
The objective of this project is to analyse daily border movement data to understand post-COVID recovery trends, seasonality, and net population movements in New Zealand.

---

## 3. Key Analysis Questions
This analysis focuses on the following questions:

1. How have border arrivals and departures changed over time?
2. Is New Zealand experiencing net population inflow or outflow during the analysis period?
3. Are there clear seasonal patterns in daily or monthly border movements?
4. Are there abnormal spikes or drops in movements that may indicate unusual events?
5. What insights can be drawn for tourism recovery and labour planning?

---

## 4. Data Source
- **Source**: Stats NZ – Daily movements across the New Zealand border  
- **Time period**: January 2024 – December 2025  
- **Data granularity**: Daily aggregated border movements
